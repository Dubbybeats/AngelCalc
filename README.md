# AngelCalc
A simple calculator made using `tkinter`. 

## Features : 

#### Version1.0

* Cross-platform
* easy to use
* supports common mathematical functions like 

| `+`   | `-`  | `*`  | `/`  |
|:-:|---|---|---|
| `.`  |  `C` | `+-`  |  `%` |
